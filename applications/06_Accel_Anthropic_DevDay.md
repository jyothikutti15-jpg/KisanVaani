## Project Name
KisanVaani — Voice-First AI Farm Advisor

## How Claude is Used (Key for this competition)

KisanVaani uses Claude in 3 powerful ways:

### 1. Claude Sonnet — Multilingual Agricultural Advisor
- System prompt with deep farming expertise across 4 countries
- Responds in same language as farmer (Hindi, Telugu, Tamil, Swahili, English)
- Personalized using farmer memory (name, crops, soil, past problems)
- Enriched with real-time context (weather, prices, schemes, alerts)
- Concise responses (80-150 words) optimized for voice delivery

### 2. Claude Vision — Crop Disease Diagnosis
- Farmer uploads photo of diseased crop
- Claude Vision analyzes leaf color, spot patterns, insect damage
- Returns diagnosis + organic/chemical treatment with exact dosages
- Tested: correctly identified Early Blight on tomato, Bacterial Blight on cotton

### 3. Claude as Expense Parser
- When farmer says "Maine 3500 rupye beej pe kharch kiye"
- Claude generates advice AND extracts structured expense JSON
- Auto-categorizes: seeds/pesticide/labor/irrigation/equipment
- Tested: 4 expenses correctly extracted from Hindi voice
