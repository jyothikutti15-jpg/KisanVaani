# IndiaAI Innovation Challenge 2026 — KisanVaani

**Website**: https://impact.indiaai.gov.in/
**Track**: Agriculture AI
**Prize Pool**: Rs 2.5 Crore for top 10
**Eligibility**: Students, researchers, professionals, startups — globally
**Status**: Watch for next cycle (2026 summit completed Feb)

---

## Sector
Agriculture

## Challenge Being Addressed
Declining crop productivity due to climate unpredictability, post-harvest losses, resource management, and access to timely market insights.

## Solution Stage
Pilot-ready (fully built, tested with real AI)

## AI Model Used
Claude (Anthropic) — Sonnet for text reasoning, Vision for photo diagnosis

## How AI is Applied
1. Natural language understanding of farmer queries in 5 languages
2. Computer vision for crop disease identification from photos
3. Contextual reasoning with farmer memory, weather, market, and scheme data
4. Auto-extraction of expense data from natural speech
5. Crop-specific reminder generation from planting events

## Impact Metrics
- 150M Indian smallholder farmers as target users
- Rs 90,000+ Crore in annual preventable crop losses addressable
- Rs 96/farmer/year operating cost (vs Rs 500+ per KVK visit)
- 20+ features, 15 unique to KisanVaani

## Scalability to 100M Users
Claude API handles unlimited concurrent requests. Add languages via config. Add states via JSON data files. PostgreSQL + horizontal scaling for production load.
