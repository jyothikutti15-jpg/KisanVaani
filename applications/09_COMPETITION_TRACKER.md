## Before Submitting Any Application

### Checklist
- [ ] Fill in [YOUR NAME], [YOUR EMAIL], [YOUR PHONE], [YOUR ORG] in the docs
- [ ] Record a 2-minute demo video (script in 00_MASTER_APPLICATION.md)
- [ ] Push code to GitHub (public repo)
- [ ] Take 3-4 screenshots of the app (Landing, Demo, Features, Alerts pages)
- [ ] Deploy to a free hosting service if possible (Render, Railway, Vercel)

### Screenshots to Take
1. **Landing page** hero section ("Every Farmer Deserves an AI Advisor")
2. **Voice Demo** showing Hindi conversation in phone simulator
3. **Features page** with weather data and yield prediction
4. **Alerts page** with country-specific alerts and community intelligence
5. **Dashboard** with call analytics charts

### Demo Video Tips
- Use OBS Studio or Windows screen recorder
- Show the phone simulator in action (Hindi → AI response)
- Upload a plant photo for diagnosis
- Switch to Kenya/Swahili
- Show the Features page (weather, yield, loans)
- End with impact numbers
- Keep it under 2 minutes
