## Project Name
KisanVaani — Voice-First AI Farm Advisor

## Category
AI for Social Good / Agriculture

## What does it do?
1.3 billion smallholder farmers have zero AI tools. KisanVaani lets any farmer call a phone number and get expert agricultural advice in their language — pest diagnosis, weather forecasts, market prices, government scheme guidance — using just a basic phone. No app, no internet needed.

## How we built it
- Claude AI (Anthropic) for multilingual agricultural reasoning + Vision for photo diagnosis
- Python FastAPI backend with 50+ API endpoints
- React + TypeScript frontend with phone simulator demo
- Open-Meteo for live weather, Twilio for phone calls
- Built 20+ features including farmer memory, crop diary, expense tracking, yield predictor, loan calculator

## What makes it unique
15 features that NO competitor has — not even India's government-backed Bharat-VISTAAR. Farmer memory across calls, photo diagnosis, auto-generated crop reminders, voice expense tracking, multi-country support (4 countries, 5 languages).
