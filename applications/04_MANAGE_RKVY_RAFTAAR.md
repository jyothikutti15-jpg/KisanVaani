**Deadline**: Check current cycle at manage.gov.in

---

## Startup Stage
Early-stage / Pre-revenue

## Sector
AgriTech — AI-powered farmer advisory

## Product Description
KisanVaani is a voice-first AI platform that provides 20+ agricultural advisory features to smallholder farmers via phone call — no smartphone or internet needed.

## What Funding Will Be Used For
1. **Cloud Infrastructure** (Rs 5L): AWS/GCP servers for production deployment
2. **Anthropic API Credits** (Rs 5L): Claude AI for 50,000+ farmer interactions
3. **Twilio Phone Numbers** (Rs 3L): Toll-free helpline for 5 Indian states
4. **Pilot Program** (Rs 7L): Field deployment in 3 districts with 10,000 farmers
5. **Team Hiring** (Rs 5L): Agriculture domain expert + field coordinator

## Traction / Validation
- Fully built product with 20+ features, all tested
- Real Claude AI responses in Hindi, Telugu, Tamil, English
- 50+ API endpoints functional
- Crop diary auto-generates 5 reminders per planting event
- Expense tracking auto-detects from Hindi voice (tested: 4 expenses, Rs 15,000)
- Live weather integration (real Open-Meteo data)

## Revenue Model
1. Government contracts: Rs 100-200/farmer/year (district/state agriculture depts)
2. Telecom partnerships: Toll-free number sponsorship
3. Agri-input advertising: "Press 1 to order neem oil from nearest shop"

## Social Impact
- 150M smallholder farmers in India lack AI tools
- Rs 90,000+ Cr in preventable crop losses annually
- KisanVaani works on Rs 500 feature phone — truly inclusive
